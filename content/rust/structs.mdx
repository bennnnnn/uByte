## Concepts in this chapter

You will learn and practice:

- **struct** — define with named fields
- **Instantiation** — Point { x: 3, y: 4 }
- **Field access** — p.x, p.y
- **Update syntax** — Point { x: 1, ..other } to copy rest from other
- **Tuple structs** — struct Color(i32, i32, i32);

---

## Defining a Struct

```rust
struct Point {
    x: i32,
    y: i32,
}

fn main() {
    let p = Point { x: 3, y: 4 };
    println!("{} {}", p.x, p.y);
}
```
