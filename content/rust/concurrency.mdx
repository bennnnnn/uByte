## Concepts in this chapter

You will learn and practice:

- **thread::spawn** — spawn a new thread; returns a JoinHandle
- **join()** — wait for the thread to finish
- **move** — move ownership into the closure for the thread
- **mpsc::channel** — multi-producer, single-consumer channel
- **Send and Sync** — trait bounds for thread safety

---

## Spawning a Thread

```rust
use std::thread;
use std::time::Duration;

fn main() {
    let handle = thread::spawn(|| {
        println!("Hello from thread");
    });
    handle.join().unwrap();
    println!("Main done");
}
```
