## Concepts in this chapter

You will learn and practice:

- **Function syntax** — `fn name(x: i32) -> i32 { }`
- **Return** — last expression without semicolon, or `return` keyword
- **Parameters** — pass by value (ownership moves for non-Copy types)
- **Multiple return** — use a tuple `(i32, bool)`
- **Unit type** — `()` for functions that don't return a value

---

## Defining Functions

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b   // no semicolon = return value
}

fn greet(name: &str) {
    println!("Hello, {}!", name);
}

fn main() {
    println!("{}", add(3, 5));
    greet("Rust");
}
```
