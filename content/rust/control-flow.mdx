## Concepts in this chapter

You will learn and practice:

- **if / else** — conditions are expressions; no parentheses required
- **if-else chain** — `else if` for multiple branches
- **match** — exhaustive pattern matching (like switch, but stronger)
- **No C-style for** — use `loop`, `while`, or `for` over iterators
- **FizzBuzz** — combine conditions in a loop

---

## if and else

In Rust, `if` is an expression — it produces a value. No parentheses around the condition:

```rust
let x = 10;
if x > 5 {
    println!("big");
}

let score = 72;
let result = if score >= 60 { "Pass" } else { "Fail" };
println!("{}", result);
```

---

## match

`match` is exhaustive: every possible value must be handled. Use `_` for a catch-all:

```rust
let day = "Monday";
let kind = match day {
    "Saturday" | "Sunday" => "Weekend",
    _ => "Weekday",
};
```
