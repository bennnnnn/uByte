## What is Rust?

Rust is a systems programming language focused on three goals: **safety**, **speed**, and **concurrency**. It achieves memory safety without a garbage collector through its unique ownership system.

**Why learn Rust?**
- **Memory safe** — no null pointers, no data races, no use-after-free
- **Blazing fast** — performance comparable to C and C++
- **Modern tooling** — `cargo` handles builds, tests, and dependencies
- **Growing demand** — used in OS kernels, WebAssembly, game engines, and CLI tools

---

## Running Rust

```bash
rustc --version      # check installed compiler
rustc hello.rs       # compile a single file
./hello              # run the binary
```

For projects, use **Cargo** (Rust's build tool and package manager):

```bash
cargo new my_project   # create a new project
cd my_project
cargo run              # compile and run
cargo build            # compile only
```

## Hello, World!

```rust
fn main() {
    println!("Hello, World!");
}
```

- `fn main()` is the entry point — every Rust binary starts here.
- `println!` is a **macro** (note the `!`). It prints a line followed by a newline.

Click **Run** above to see the output.

## Variables

```rust
let language = "Rust";   // immutable by default
let version: u32 = 1;
println!("Learning {} version {}", language, version);
```

Variables are **immutable by default**. Add `mut` to make one mutable:

```rust
let mut count = 0;
count += 1;
println!("{}", count); // 1
```

---

## Format Strings

Use `{}` as a placeholder in `println!` and `format!`:

```rust
let name = "Learner";
let greeting = format!("Hello, {}!", name);
println!("{}", greeting); // Hello, Learner!
```
