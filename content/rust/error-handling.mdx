## Concepts in this chapter

You will learn and practice:

- **Result&lt;T, E&gt;** — Ok(value) or Err(error)
- **Option&lt;T&gt;** — Some(value) or None
- **unwrap** — panic on Err/None (use for quick tests)
- **? operator** — propagate errors to caller (in functions returning Result)
- **match** — handle Ok/Err or Some/None explicitly

---

## Result

```rust
fn parse(s: &str) -> Result<i32, std::num::ParseIntError> {
    s.parse()
}

fn main() {
    match parse("42") {
        Ok(n) => println!("{}", n),
        Err(e) => println!("Invalid: {}", e),
    }
}
```

---

## unwrap and expect

```rust
let n: i32 = "42".parse().unwrap();
let n = "x".parse().expect("not a number");  // panic with message
```
