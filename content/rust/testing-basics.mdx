## Concepts in this chapter

You will learn and practice:

- **#[test]** — mark a function as a test
- **assert!** — assert!(condition)
- **assert_eq!** — assert_eq!(expected, actual)
- **#[cfg(test)]** — compile tests only when testing
- **cargo test** — run all tests

---

## Simple Test

```rust
#[cfg(test)]
mod tests {
    #[test]
    fn it_works() {
        assert_eq!(2 + 2, 4);
    }
}
```

In the exercise we simulate the test by calling the function and printing pass/fail so the runner can check output.
