## Defining Functions

```cpp
int add(int a, int b) {
    return a + b;
}

int main() {
    std::cout << add(3, 4) << std::endl; // 7
}
```

## Default Arguments

```cpp
void greet(std::string name = "World") {
    std::cout << "Hello, " << name << "!" << std::endl;
}
```
