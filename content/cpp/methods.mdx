## Member Functions

```cpp
class Rectangle {
public:
    int width, height;
    Rectangle(int w, int h) : width(w), height(h) {}
    int area() const { return width * height; }
};
Rectangle r(4, 5);
std::cout << r.area() << std::endl; // 20
```
