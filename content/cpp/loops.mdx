## for Loop

```cpp
for (int i = 0; i < 5; i++) {
    std::cout << i << std::endl;
}
```

## Range-based for (C++11)

```cpp
std::vector<int> nums = {1, 2, 3};
for (auto n : nums) {
    std::cout << n << std::endl;
}
```

## while Loop

```cpp
int i = 0;
while (i < 5) {
    std::cout << i++ << std::endl;
}
```
