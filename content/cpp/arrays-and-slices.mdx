## std::vector (dynamic array)

```cpp
#include <vector>
std::vector<int> v = {1, 2, 3};
v.push_back(4);
std::cout << v.size() << std::endl; // 4
```

## std::array (fixed size)

```cpp
#include <array>
std::array<int, 3> a = {10, 20, 30};
std::cout << a[0] << std::endl; // 10
```
