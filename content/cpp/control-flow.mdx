## if / else if / else

```cpp
int x = 10;
if (x > 0) {
    std::cout << "positive" << std::endl;
} else if (x < 0) {
    std::cout << "negative" << std::endl;
} else {
    std::cout << "zero" << std::endl;
}
```

## switch

```cpp
int day = 2;
switch (day) {
    case 1: std::cout << "Monday"; break;
    case 2: std::cout << "Tuesday"; break;
    default: std::cout << "Other"; break;
}
```
