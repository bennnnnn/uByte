## Abstract Class (Interface)

```cpp
class Shape {
public:
    virtual double area() const = 0; // pure virtual
    virtual ~Shape() {}
};

class Circle : public Shape {
    double r;
public:
    Circle(double r) : r(r) {}
    double area() const override { return 3.14159 * r * r; }
};
```

Use `= 0` to make a function pure virtual (must be overridden). A class with any pure virtual is abstract.
