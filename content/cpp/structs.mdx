## struct

```cpp
struct Point {
    double x;
    double y;
};
Point p = {3.0, 4.0};
std::cout << p.x << std::endl;
```

## class with Constructor

```cpp
class Circle {
public:
    double radius;
    Circle(double r) : radius(r) {}
    double area() { return 3.14159 * radius * radius; }
};
```
