## std::map

```cpp
#include <map>
std::map<std::string, int> ages;
ages["Alice"] = 30;
ages["Bob"] = 25;
std::cout << ages["Alice"] << std::endl; // 30
```

## std::unordered_map (O(1) average lookup)

```cpp
#include <unordered_map>
std::unordered_map<std::string, int> scores;
scores["Alice"] = 95;
```
