## Header Files

```cpp
// math_utils.h
#pragma once
int add(int a, int b);

// math_utils.cpp
#include "math_utils.h"
int add(int a, int b) { return a + b; }
```

## Namespaces

```cpp
namespace myapp {
    void hello() { std::cout << "Hello from myapp!" << std::endl; }
}
myapp::hello();
```
