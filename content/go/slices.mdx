## What Is a Slice Used For?

A slice stores an ordered list of values, just like an array, but a slice is **flexible**.

You use a slice when:

- you want a list of values in order
- the number of values may change
- you want to loop through many items

Examples:

- a list of lesson names
- a list of scores
- a list of products in a cart

In Go, slices are much more common than arrays in day-to-day code.

---

## Declaring a Slice

Use `[]type` to declare a slice:

```gorun
package main

import "fmt"

func main() {
    names := []string{"Ana", "Ben", "Cory"}
    var scores []int

    fmt.Println(names)
    fmt.Println(scores)
}
```

Two common patterns:

- `names := []string{"Ana", "Ben", "Cory"}` creates a slice with values
- `var scores []int` declares an empty slice

---

## Assigning and Updating Values

You can change a slice element by index.

```gorun
package main

import "fmt"

func main() {
    scores := []int{80, 85, 90}
    scores[1] = 95

    fmt.Println(scores)
}
```

Indexes start at `0`, so:

- `scores[0]` is the first value
- `scores[1]` is the second value
- `scores[2]` is the third value

---

## Accessing the First and Last Value

```gorun
package main

import "fmt"

func main() {
    nums := []int{10, 20, 30, 40, 50}

    fmt.Println("First:", nums[0])
    fmt.Println("Last:", nums[len(nums)-1])
}
```

The last index is always `len(slice)-1`.

---

## Loop Through Every Value

Use a loop when you want to print every item:

```gorun
package main

import "fmt"

func main() {
    nums := []int{3, 6, 9, 12}

    for i := 0; i < len(nums); i++ {
        fmt.Println(nums[i])
    }
}
```

This is the clearest beginner pattern because you can see the index and the value access together.

---

## Find the Minimum and Maximum

To find the smallest and largest values, start with the first item and compare the rest:

```gorun
package main

import "fmt"

func main() {
    nums := []int{8, 3, 15, 2, 10}
    min := nums[0]
    max := nums[0]

    for i := 1; i < len(nums); i++ {
        if nums[i] < min {
            min = nums[i]
        }
        if nums[i] > max {
            max = nums[i]
        }
    }

    fmt.Println("Min:", min)
    fmt.Println("Max:", max)
}
```

This is a very common slice pattern in real programs.

---

## Sum All Elements

You can also add all values in a slice:

```gorun
package main

import "fmt"

func main() {
    nums := []int{1, 2, 3, 4, 5}
    sum := 0

    for i := 0; i < len(nums); i++ {
        sum += nums[i]
    }

    fmt.Println(sum)
}
```

This pattern appears often for totals, averages, and score calculations.

---

## A Quick Look at `append`

One reason slices are so useful is that you can grow them:

```gorun
package main

import "fmt"

func main() {
    nums := []int{1, 2, 3}
    nums = append(nums, 4)

    fmt.Println(nums)
}
```

You do not need to master every slice feature yet. For now, focus on reading, updating, looping, and summing.
