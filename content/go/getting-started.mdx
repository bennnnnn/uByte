## What Is Go, and Why Should You Care?

Go (also called Golang) is a programming language created by Google in 2009. It was designed to be **fast to run** and **fast to write** — engineers who were frustrated with slow build times and complicated code invented Go to fix those problems.

Today, Go powers some of the world's busiest systems: Docker, Kubernetes, Cloudflare, Dropbox, and Uber all use it heavily. It's also one of the highest-paying languages to know.

In this course, you'll build a **student tracker for Riverside Academy** — starting here, adding features in every chapter. By the end, you'll have written a real Go program from scratch.

## The Structure of Every Go Program

Open any Go file and you'll see the same two things at the top:

```go
package main

import "fmt"

func main() {
    // your code goes here
}
```

- `package main` — tells Go this file is an executable program (not a library)
- `import "fmt"` — loads the `fmt` package, which handles printing
- `func main()` — the entry point; Go runs this function first when you start the program

Every single Go program you write in this course will have this structure. Think of it as the envelope that holds your code.

## Printing Output

The `fmt` package gives you three tools for output:

### Println — simple, automatic newline

```gorun
package main

import "fmt"

func main() {
    fmt.Println("Hello, Riverside Academy!")
    fmt.Println("Welcome to the student tracker.")
}
```

Each `Println` call prints one line and automatically moves to the next. Run it and see.

### Printf — templates with placeholders

`fmt.Printf` lets you build a message by filling in placeholders called **format verbs**:

| Verb | Meaning |
|------|---------|
| `%s` | a string |
| `%d` | a whole number (integer) |
| `%f` | a decimal number (float) |
| `\n` | move to the next line |

```gorun
package main

import "fmt"

func main() {
    school := "Riverside Academy"
    year := 2025
    fmt.Printf("Welcome to %s, %d\n", school, year)
}
```

### Sprintf — build a string instead of printing it

`fmt.Sprintf` works exactly like Printf but **returns** the formatted string so you can store and reuse it:

```gorun
package main

import "fmt"

func main() {
    id := 1042
    name := "Alice"
    badge := fmt.Sprintf("Student ID: %d — %s", id, name)
    fmt.Println(badge)
}
```

## Comments — Notes for Humans

Comments start with `//` and are completely ignored by Go. They exist only for the people reading your code:

```go
// This is a comment — Go ignores it completely
fmt.Println("This runs fine")
```

Use comments to explain *why* you made a decision, not just what the code does. In this course, every step will remind you when a comment is helpful.
