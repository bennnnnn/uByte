## What Is an Array?

An array is an ordered collection with a **fixed size**.

If you create an array of 5 integers, it always has room for exactly 5 integers. You can change the values inside it, but you cannot grow or shrink it.

Arrays are useful when:

- you know exactly how many values you need
- the size should not change
- you want simple indexed storage like positions 0, 1, 2, and so on

```gorun
package main

import "fmt"

func main() {
    var scores [3]int
    fmt.Println(scores) // [0 0 0]
}
```

Go fills a new array with zero values.

## Declaring Arrays

Use `[size]type` to declare an array:

```gorun
package main

import "fmt"

func main() {
    var numbers [4]int
    names := [3]string{"Ana", "Ben", "Cory"}
    colors := [...]string{"red", "green", "blue"}

    fmt.Println(numbers)
    fmt.Println(names)
    fmt.Println(colors)
}
```

Three common patterns:

- `var numbers [4]int` creates an array with zero values
- `[3]string{"Ana", "Ben", "Cory"}` creates and fills the array
- `[...]string{...}` lets Go count the size for you

## Assigning Values by Index

Array indexes start at `0`.

```gorun
package main

import "fmt"

func main() {
    var scores [3]int

    scores[0] = 90
    scores[1] = 85
    scores[2] = 100

    fmt.Println(scores)
}
```

Index positions here are:

- `scores[0]` -> first value
- `scores[1]` -> second value
- `scores[2]` -> third value

---

## Reading the First and Last Value

Use an index to read values from the array.

```gorun
package main

import "fmt"

func main() {
    temps := [4]int{18, 21, 24, 19}

    fmt.Println("First:", temps[0])
    fmt.Println("Last:", temps[len(temps)-1])
    fmt.Println("Length:", len(temps))
}
```

`len(temps)-1` gives the last valid index.

---

## Loop Through an Array

You can use a loop to print every value one at a time.

```gorun
package main

import "fmt"

func main() {
    nums := [4]int{2, 4, 6, 8}

    for i := 0; i < len(nums); i++ {
        fmt.Println(nums[i])
    }
}
```

This pattern is useful when you want full control over the index.

---

## Sum the Elements

Arrays are often used with a running total:

```gorun
package main

import "fmt"

func main() {
    nums := [5]int{1, 2, 3, 4, 5}
    sum := 0

    for i := 0; i < len(nums); i++ {
        sum += nums[i]
    }

    fmt.Println(sum)
}
```

This same idea will carry into the next lesson on slices.
