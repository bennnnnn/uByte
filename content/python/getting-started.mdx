## What is Python?

Python is a high-level, interpreted programming language created by Guido van Rossum and first released in 1991. It emphasizes readability and simplicity — perfect for beginners, scripting, data science, and web development.

**Why learn Python?**
- **Readable syntax** — clean and minimal, often called "executable pseudocode"
- **Large standard library** — batteries included for file I/O, HTTP, JSON, and more
- **Huge ecosystem** — PyPI has hundreds of thousands of packages
- **Used everywhere** — Google, Netflix, Instagram, NASA, and most data/ML teams

---

## Installing Python

Visit [python.org/downloads](https://www.python.org/downloads/) and download the latest Python 3 installer:

- **macOS / Linux:** Often pre-installed; check with `python3 --version`. Or use the official installer or your package manager.
- **Windows:** Download the installer and check "Add Python to PATH".

After installation, open your terminal and verify:

```bash
python3 --version
```

You should see something like `Python 3.12.0`. Use `python3` (and `pip3`) on macOS/Linux if both Python 2 and 3 are installed.

## Your First Python Program: Hello World

Create a file called `hello.py` and add:

```python
print("Hello, World!")
```

Click **Run** above to see the output. In Python there is no `main` function required — the script runs from top to bottom. The `print()` function sends text to the console.

---

## Printing and Formatting Output

You can call `print()` many times; each call prints one line by default. Use f-strings to embed values:

```python
name = "Python"
version = 3
print(f"Learning {name} version {version}")
```

**Quick reference:**
- `print("text")` — prints with a newline
- `print(a, b)` — prints multiple values separated by spaces
- `print(f"{x}")` — f-string formatting

---

## Comments

Comments start with `#` and are ignored by Python:

```python
# This is a comment
print("Comments are ignored")  # inline comment
```
