## Arrays

Fixed-size, zero-indexed:

```csharp
int[] nums = { 10, 20, 30, 40, 50 };
Console.WriteLine(nums[0]);        // 10
Console.WriteLine(nums.Length);    // 5
nums[2] = 99;
```

Multi-dimensional:

```csharp
int[,] grid = new int[3, 3];
grid[0, 0] = 1;

// Jagged (array of arrays)
int[][] jagged = new int[3][];
jagged[0] = new[] { 1, 2 };
```

## List\<T\>

Dynamic size — grows automatically:

```csharp
using System.Collections.Generic;

var fruits = new List<string> { "apple", "banana" };
fruits.Add("cherry");
fruits.Remove("apple");
Console.WriteLine(fruits.Count);    // 2
Console.WriteLine(fruits[0]);       // banana

foreach (var f in fruits) {
    Console.WriteLine(f);
}
```

## Useful Array Methods

```csharp
int[] nums = { 3, 1, 4, 1, 5 };
Array.Sort(nums);                  // in-place sort
Array.Reverse(nums);               // in-place reverse
int idx = Array.IndexOf(nums, 4);  // find index
```

---

## Useful List Methods

```csharp
var list = new List<int> { 1, 2, 3 };
list.Insert(0, 0);           // insert at index
list.RemoveAt(2);            // remove at index
bool has = list.Contains(1); // true
list.Sort();
list.Clear();
```

## Span\<T\> — Zero-Allocation Slices

```csharp
int[] arr = { 1, 2, 3, 4, 5 };
Span<int> slice = arr.AsSpan(1, 3);  // [2, 3, 4]
```
