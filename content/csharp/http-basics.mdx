## HttpClient

The standard way to make HTTP requests in .NET:

```csharp
using System.Net.Http;
using System.Net.Http.Json;

// Create once and reuse (avoid creating per-request — it exhausts sockets)
using var client = new HttpClient {
    BaseAddress = new Uri("https://jsonplaceholder.typicode.com"),
};
```

---

## GET Request

```csharp
var response = await client.GetAsync("/posts/1");
response.EnsureSuccessStatusCode();          // throws on 4xx/5xx

string json = await response.Content.ReadAsStringAsync();
Console.WriteLine(json);
```

Deserialize JSON directly:

```csharp
var post = await client.GetFromJsonAsync<Post>("/posts/1");
Console.WriteLine(post?.Title);
```

---

## POST Request

```csharp
var newPost = new { title = "Hello", body = "World", userId = 1 };
var response = await client.PostAsJsonAsync("/posts", newPost);
response.EnsureSuccessStatusCode();

var created = await response.Content.ReadFromJsonAsync<Post>();
Console.WriteLine(created?.Id);
```

---

## Setting Headers

```csharp
client.DefaultRequestHeaders.Authorization =
    new AuthenticationHeaderValue("Bearer", "your-token");

client.DefaultRequestHeaders.Accept.Add(
    new MediaTypeWithQualityHeaderValue("application/json"));
```

---

## Response Status Codes

```csharp
var response = await client.GetAsync("/items/999");

if (response.StatusCode == System.Net.HttpStatusCode.NotFound) {
    Console.WriteLine("Not found");
} else if (!response.IsSuccessStatusCode) {
    Console.WriteLine($"Error: {(int)response.StatusCode}");
}
```

---

## IHttpClientFactory (Production Pattern)

Register in DI and inject — avoids socket exhaustion:

```csharp
// In Program.cs / Startup.cs
builder.Services.AddHttpClient<IMyApiClient, MyApiClient>(c => {
    c.BaseAddress = new Uri("https://api.example.com");
});
```
