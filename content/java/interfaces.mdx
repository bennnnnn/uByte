## Concepts in this chapter

You will learn and practice:

- **Interface** — contract of methods; class implements interface
- **Abstract class** — can have abstract methods and concrete methods
- **Polymorphism** — reference type is interface; runtime type is implementation
- **@Override** — override interface or parent methods

---

## Interface

```java
public interface Drawable {
    void draw();
}

public class Circle implements Drawable {
    @Override
    public void draw() {
        System.out.println("Drawing a circle");
    }
}

Drawable d = new Circle();
d.draw();
```

---

## Abstract Class

```java
public abstract class Shape {
    public abstract double area();
    public void describe() {
        System.out.println("Area: " + area());
    }
}
```
