## Concepts in this chapter

You will learn and practice:

- **package** — declare at top of file; matches directory structure
- **import** — use classes from java.util, java.io, or your own packages
- **Static import** — import static for constants and static methods
- **Fully qualified name** — use full name without import (e.g. java.util.ArrayList)

---

## Package Declaration

```java
package com.example.app;

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello");
    }
}
```

File must live in `com/example/app/Main.java`.

---

## Imports

```java
import java.util.ArrayList;
import java.util.HashMap;
import java.util.Map;

// or single type
import java.util.Scanner;

// static import
import static java.lang.Math.PI;
import static java.lang.Math.sqrt;
```
