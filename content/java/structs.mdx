## Concepts in this chapter

You will learn and practice:

- **Class definition** — fields and methods
- **Constructors** — same name as class; no return type
- **new** — create an object; reference type
- **Encapsulation** — private fields, public getters/setters
- **this** — refer to the current object

---

## Simple Class

```java
public class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() { return name; }
    public int getAge() { return age; }
}

// usage:
Person p = new Person("Alice", 30);
System.out.println(p.getName());
```

---

## Constructor and Fields

- The constructor runs when you call `new Person(...)`.
- `this.name` refers to the instance field (not the parameter).
