## Concepts in this chapter

You will learn and practice:

- **JUnit 5** — @Test, assertions (assertEquals, assertTrue)
- **Test class** — public class with public void test methods
- **Assertions** — assertEquals(expected, actual), assertTrue(condition)
- **Running tests** — IDE or mvn test / gradle test

---

## Simple Test

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.assertEquals;

public class CalculatorTest {
    @Test
    void add() {
        assertEquals(5, 2 + 3);
    }
}
```

---

## Assertions

- **assertEquals(expected, actual)** — must be equal
- **assertTrue(boolean)** — condition is true
- **assertThrows** — expect an exception
