## Concepts in this chapter

You will learn and practice:

- **JSON format** — objects {}, arrays [], strings, numbers
- **Manual parsing** — for simple cases; use libraries in production
- **Gson / Jackson** — fromJson, toJson; ObjectMapper
- **POJOs** — plain Java objects with getters/setters for mapping
- **JSON in HTTP** — send and receive JSON in request/response body

---

## Simple JSON String

```java
// Without a library, build JSON manually for simple cases:
String name = "Alice";
int age = 30;
String json = "{\"name\":\"" + name + "\",\"age\":" + age + "}";
System.out.println(json);
```
