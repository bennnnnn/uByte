## Concepts in this chapter

You will learn and practice:

- **Thread** — create a thread with Runnable or extending Thread
- **start()** — begin execution; run() is called by the JVM
- **join()** — wait for a thread to finish
- **Runnable** — functional interface; can use lambda
- **Concurrency** — multiple threads; shared state needs care

---

## Creating a Thread

```java
Thread t = new Thread(() -> {
    System.out.println("Hello from thread");
});
t.start();
t.join();  // wait for thread to finish
```

---

## Runnable Lambda

```java
Runnable task = () -> System.out.println("Running");
new Thread(task).start();
```
