## Java's Type System

Java is **statically and strongly typed**. Every variable must have a declared type (or use `var` for local inference). The compiler checks types before your program runs.

---

## Primitive Types

| Type | Size | Example |
|------|------|---------|
| `int` | 32-bit | `int age = 30;` |
| `long` | 64-bit | `long pop = 8_000_000_000L;` |
| `double` | 64-bit | `double pi = 3.14159;` |
| `boolean` | 1-bit | `boolean active = true;` |
| `char` | 16-bit | `char grade = 'A';` |

```java
int age = 30;
double pi = 3.14159;
boolean active = true;
System.out.println(age);    // 30
System.out.println(pi);     // 3.14159
System.out.println(active); // true
```

---

## String

`String` is a class, not a primitive. Strings are **immutable** in Java.

```java
String name = "Alice";
System.out.println(name.toUpperCase());   // ALICE
System.out.println(name.length());        // 5
System.out.println(name.substring(0, 3)); // Ali
```

---

## var (Java 10+)

Use `var` for local variable type inference:

```java
var city = "London";       // String
var population = 9_000_000; // int
System.out.println(city);
System.out.println(population);
```

`var` only works for local variables, not for fields or method parameters.

---

## Type Casting

Widening (int → double) is automatic. Narrowing (double → int) requires an explicit cast:

```java
int i = 42;
double d = i;          // widening — automatic
int back = (int) 3.99; // narrowing — explicit, truncates to 3

System.out.println(d);    // 42.0
System.out.println(back); // 3
```

Convert to/from String:

```java
String s = String.valueOf(42);  // int → String
int n = Integer.parseInt("99"); // String → int
```

---

## Constants with final

`final` prevents reassignment:

```java
final double PI = 3.14159;
double radius = 5.0;
double area = PI * radius * radius;
System.out.println(String.format("%.2f", area)); // 78.54
```

By convention, constants use `UPPER_SNAKE_CASE`.
