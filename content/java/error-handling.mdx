## Concepts in this chapter

You will learn and practice:

- **try-catch** — catch exceptions; prevent program crash
- **throw** — throw checked or unchecked exceptions
- **finally** — run cleanup code whether or not an exception occurred
- **Checked vs unchecked** — IOException is checked; RuntimeException is not
- **Exception hierarchy** — Exception, RuntimeException, Error

---

## try and catch

```java
try {
    int n = Integer.parseInt("abc");  // throws NumberFormatException
} catch (NumberFormatException e) {
    System.out.println("Invalid number: " + e.getMessage());
}
```

---

## throw and finally

```java
if (age < 0) {
    throw new IllegalArgumentException("Age cannot be negative");
}

try {
    // ...
} finally {
    System.out.println("Cleanup");
}
```
