## Concepts in this chapter

---

## HashMap Basics

```java
import java.util.HashMap;
import java.util.Map;

Map<String, Integer> ages = new HashMap<>();
ages.put("Alice", 30);
ages.put("Bob", 25);
System.out.println(ages.get("Alice"));   // 30
System.out.println(ages.containsKey("Bob"));  // true
```

---

## getOrDefault and putIfAbsent

```java
int age = ages.getOrDefault("Charlie", 0);  // 0 if missing
ages.putIfAbsent("Diana", 28);  // put only if key absent
```

---

## Iterating

```java
for (String name : ages.keySet()) {
    System.out.println(name + ": " + ages.get(name));
}
for (Map.Entry<String, Integer> e : ages.entrySet()) {
    System.out.println(e.getKey() + " -> " + e.getValue());
}
```
