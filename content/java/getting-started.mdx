## What is Java?

Java is a class-based, object-oriented language created by Sun Microsystems in 1995. It runs on the **Java Virtual Machine (JVM)**, making it platform-independent — "write once, run anywhere."

**Why learn Java?**
- **Widespread** — used in Android, enterprise backends, big data, and more
- **Strongly typed** — catch bugs at compile time
- **Rich standard library** — batteries included
- **Long-term support** — Java 21 (LTS) is widely deployed

---

## Running Java

```bash
java --version       # check installed JVM
javac Hello.java     # compile to bytecode
java Hello           # run the bytecode
```

With modern tooling you can also use `java Hello.java` to run directly (Java 11+).

## Hello, World!

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

Every Java program needs at least one **class** whose name matches the file name. Execution begins in `public static void main(String[] args)`.

`System.out.println()` prints a line followed by a newline.

## Variables

```java
String language = "Java";
int version = 21;
System.out.println("Learning " + language + " version " + version);
```

Java is **statically typed** — every variable has a fixed type declared at compile time. Since Java 10 you can use `var` for local type inference:

```java
var city = "London";   // inferred as String
var count = 42;        // inferred as int
```

---

## String Formatting

```java
String name = "Learner";
System.out.printf("Hello, %s!%n", name);         // printf style
String msg = String.format("Hello, %s!", name);   // build string
System.out.println(msg);
```
