## Class

```javascript
class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(`${this.name} makes a sound.`);
  }
}
const a = new Animal("Dog");
a.speak(); // Dog makes a sound.
```

## Inheritance

```javascript
class Dog extends Animal {
  speak() {
    console.log(`${this.name} barks.`);
  }
}
```

## Private Fields (ES2022)

```javascript
class Counter {
  #count = 0;
  increment() { this.#count++; }
  get() { return this.#count; }
}
```
