## Primitives — Passed by Value

```javascript
let a = 10;
let b = a;
b = 20;
console.log(a); // 10 — unchanged
```

## Objects — Passed by Reference

```javascript
const obj = { x: 1 };
const ref = obj;
ref.x = 99;
console.log(obj.x); // 99 — same object!
```

## Cloning Objects

```javascript
const copy = { ...obj };        // shallow clone
const deep = structuredClone(obj); // deep clone (modern)
```
