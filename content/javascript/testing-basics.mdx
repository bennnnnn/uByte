## Jest

```javascript
// math.test.js
import { add } from "./math.js";

describe("add function", () => {
  it("adds two positive numbers", () => {
    expect(add(2, 3)).toBe(5);
  });
  it("handles negatives", () => {
    expect(add(-1, 1)).toBe(0);
  });
});
```

Run with `npx jest` or `npm test`.
