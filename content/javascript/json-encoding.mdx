## JSON.stringify (object → string)

```javascript
const user = { name: "Alice", age: 30, city: "London" };
const json = JSON.stringify(user);
console.log(json); // {"name":"Alice","age":30,"city":"London"}

// Pretty-print
console.log(JSON.stringify(user, null, 2));
```

## JSON.parse (string → object)

```javascript
const parsed = JSON.parse('{"name":"Alice","age":30}');
console.log(parsed.name); // Alice
```

## Handling Dates and Special Values

```javascript
const obj = { date: new Date().toISOString(), score: null };
const json = JSON.stringify(obj);
```
