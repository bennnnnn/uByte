## if / else if / else

```javascript
const x = 10;
if (x > 0) {
  console.log("positive");
} else if (x < 0) {
  console.log("negative");
} else {
  console.log("zero");
}
```

## Ternary Operator

```javascript
const label = x > 0 ? "positive" : "non-positive";
```

## Nullish Coalescing (??)

```javascript
const name = null ?? "Guest"; // "Guest"
```

## switch

```javascript
switch (day) {
  case 1: console.log("Monday"); break;
  case 2: console.log("Tuesday"); break;
  default: console.log("Other");
}
```
