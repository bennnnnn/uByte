## for Loop

```javascript
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

## for...of (arrays)

```javascript
const nums = [1, 2, 3];
for (const n of nums) {
  console.log(n);
}
```

## while Loop

```javascript
let i = 0;
while (i < 5) {
  console.log(i++);
}
```

## Array forEach

```javascript
[1, 2, 3].forEach(n => console.log(n));
```
