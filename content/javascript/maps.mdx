## Plain Object

```javascript
const person = { name: "Alice", age: 30 };
console.log(person.name);        // Alice
console.log(person["age"]);      // 30
person.city = "London";          // add property
```

## Map (key-value, any key type)

```javascript
const map = new Map();
map.set("key", 42);
console.log(map.get("key")); // 42
console.log(map.size);       // 1
```
