## console.log

`console.log()` accepts any number of arguments, printing them space-separated:

```javascript
console.log("Name:", "Alice", "Age:", 30);
// Name: Alice Age: 30
```

---

## Template Literals

The JavaScript equivalent of `fmt.Sprintf` — embed expressions with `${...}`:

```javascript
const name = "Alice";
const age = 30;
console.log(`Name: ${name}, Age: ${age}`);
```

---

## Number Formatting

```javascript
const pi = 3.14159;
console.log(pi.toFixed(2));       // "3.14"
console.log(pi.toPrecision(4));   // "3.142"
console.log((1234567).toLocaleString()); // "1,234,567"
```

---

## Building Strings

Use template literals or string concatenation:

```javascript
const host = "localhost";
const port = 8080;
const url = `http://${host}:${port}/api`;
console.log(url); // http://localhost:8080/api
```

---

## console Methods

```javascript
console.log("info");        // standard output
console.error("error!");    // stderr (red in terminals)
console.warn("warning");    // warning
console.table([{a:1},{a:2}]); // formatted table
```
